# SiSiC_Fusion_SwitchingEnergyShareRate
Supporting data for T. Reiter, M. Niendorf, J. Schapdick, "A Method for Individual Switching Energy Extraction in Si/SiC Fusion Switches", ECCE Europe, 2026 (planned publication)
For more detailed description please refer to the paper!

# File Description:
* ExampleData.mat: Some measured scope waveforms as example
* Energy_ONFUSION.m: Function to extract operating points, dv/dts, di/dts and the energy share rates between IGBT and SiC MOSFET
* Run_Example.m: Load the example and run the function
